# Kivalliq Arctic charr lcWGS pipeline

Population genomics pipeline for Chapter 1 (CJFAS manuscript) -- fine-scale
regional structure and Atlantic-Arctic introgression in anadromous Arctic
charr (*Salvelinus alpinus*) across 13 populations (405 individuals) in the
Kivalliq region, Nunavut.

This repository contains **only** the scripts behind the figures, tables,
and analyses reported in the manuscript. Exploratory analyses that were run
during the project but did not make it into the final article (environmental
RDA, dispersal assignment, D-stat pairwise significance heatmaps, etc.) are
not included.

## Requirements

- SLURM cluster with ANGSD, ngsParalog, ngsLD, NGSadmix, PCAngsd installed
- R with `data.table`, `ggplot2`, `dplyr`, `tidyr`, `patchwork`, `vegan`,
  `readr`, `reshape2`, `scales`, `stringr`, `viridisLite`, `sf`,
  `ggspatial`, `ggrepel`, `GenomicRanges`, `marmap`
  (marmap only needed for `compute_hydro_distances.R`, in its own env)
- Python 3 (utility scripts, standard library only)

**Important:** point `RSCRIPT` in `config/00_config.sh` at an R installation
that has `nlme` and `vegan` installed together without conflict. A generic
"system R" module can silently break the Mantel/IBD step — use a dedicated
conda/renv environment.

## Setup

1. Edit `config/00_config.sh` — fill in the **USER-SPECIFIC PATHS** section
   at the top (pipeline root, reference genome, existing BAM/metadata
   source, program binaries, R installation).
2. Run `01_scripts/bash/00_setup.sh` once to create the working directory
   tree and pull in your BAM list / population / region metadata.
3. Place bamlists for the three reference populations at
   `${INFO_DIR}/bamlists/{JAY,LLS,DV}.bamlist` — these are not part of the
   13 Kivalliq populations and are not generated by `00_setup.sh`:
   - **JAY** — pure Beringian Arctic reference (Kitikmeot)
   - **LLS** — pure Atlantic reference (Lake Lilla Stensjön, Sweden;
     Saha et al. 2024, *Conserv Genet* 25:513–531)
   - **DV** — Dolly Varden (*S. malma*, Babbage River, Yukon), used as the
     D-statistics outgroup (H4)
4. Basemap shapefiles used by the mapping figures (`plot_panel_*_structure.R`,
   `plot_management_units.R`) are **not included** in this repo. They are
   Statistics Canada administrative boundaries (`lpr_000b21a_e.shp`) and
   NRCan CanVec hydrography (`waterbody_2.shp`, `watercourse_1.shp`).
   Download them separately and pass the containing directory as the
   `shp_dir` argument to those scripts.

## Reference genome

`GENOME` should point at GCF_002910315.2 / ASM291031v2 (Christensen et al.
2018). **Note:** this genome assembly's publication has been retracted
(the sequenced individual is now thought to be *S. malma* or a hybrid); the
assembly itself is unaffected and all samples here use the same reference,
so relative comparisons are not compromised. This is disclosed in the
manuscript's Methods and Limitations and should be disclosed by anyone
reusing this reference.

## K selection (admixture)

Optimal K (global = 7, Rankin Inlet = 2, Naujaat = 5) was **not** chosen by
a script in this repo. Per-replicate NGSadmix ancestry matrices and
log-likelihoods were uploaded to the **StructureSelector** web tool, which
computes the Puechmaille (2016) estimators (MedMedK, MedMeaK, MaxMedK,
MaxMeaK) alongside the fastSTRUCTURE Choose-K criterion. These were
preferred over the classic Evanno ΔK, which is biased under the unbalanced
per-population sample sizes here (14–41 individuals). See manuscript
Methods ("Admixture analysis") for details.

| 03 | `03A_ngsLD.sh` → `03B_ld_decay.sh` → `03C_ld_pruning.sh` → `03D_concat_pruned.sh` | LD estimation, decay, pruning | — |
| 04 | `04_pca.sh` | PCAngsd (global, Rankin, Naujaat) | Fig. 2b, 3c-d |
| 05 | `05_admixture.sh` | NGSadmix (global, Rankin, Naujaat); K chosen via StructureSelector (see above) | Fig. 2c, 3e-f |
| 06 | `06_fst.sh` | Pairwise FST (78 pairs) + sliding window | Fig. S1, S2; Fig. 6b |
| 07 | `07_thetas.sh` | Nucleotide diversity (π, θW, Tajima's D) | Table 2; Fig. 6c |
| 08 | `08_mask_deviants.sh` | Masked ancestral genome (feeds into 07) | — |
| 09 | `09_ibd.sh` | Isolation-by-distance (Mantel); needs `compute_hydro_distances.R` run first | Fig. 6e |

| 11 | `11_dstats.sh` → `11B_dstats_figures.sh` | D-statistics D(JAY, P; LLS, DV) | Fig. 5, 6d; Table 3 |

Figures assembled directly from upstream tables (run manually with `$RSCRIPT`,
not submitted via `sbatch`):
- `plot_panel_global_structure.R <base_dir> <shp_dir>` — Fig. 1 (map + inset) + Fig. 2 (global PCA/admixture)
- `plot_panel_regional_structure.R <base_dir> <shp_dir>` — Fig. 3 (regional PCA/admixture)
- `plot_management_units.R <base_dir> <shp_dir>` — Fig. 6 (synthesis / management units)
